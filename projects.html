Nike Shoe Application

This application allows users to browse, select, and purchase Nike shoes online. It features an intuitive interface where users can view detailed information about various Nike shoe models, including images, prices, and reviews. Users can add shoes to their cart, track their orders, and manage their profiles.

Technologies Used: React, Node.js, Express, MongoDB, Stripe API for payments, and Bootstrap for styling.

Achievements: Successfully implemented a secure payment system with Stripe. Designed a responsive and user-friendly interface that adapts to various screen sizes. Integrated real-time inventory management to ensure accurate stock levels.
